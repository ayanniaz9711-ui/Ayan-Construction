# 🏗️ Ayan Construction

A modern and responsive construction company website built with HTML, CSS, and JavaScript. The website showcases construction services, completed projects, company expertise, and contact information with a clean and professional design.

## 🌐 Live Demo
https://ayan-construction.netlify.app/

## ✨ Features

- Modern & Professional UI
- Fully Responsive Design
- Construction Services Section
- Project Showcase
- About Company
- Contact Form
- Smooth Navigation
- Mobile Friendly

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Ayan-Construction/
│── index.html
│── style.css
│── script.js
│── images/
└── README.md
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ayan-construction.git
```

2. Open `index.html` in your browser.
